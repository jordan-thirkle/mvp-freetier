# FreeTier

> **Bootstrapper's catalog of free SaaS tools.**

Discover free tiers, generous plans, and cheap alternatives for indie SaaS builders. **Hosting, databases, auth, email, analytics, AI/LLMs, payments, monitoring, CMS** — build your entire SaaS stack without spending a cent.

## Features

- **Curated Catalog** — 150+ tools across 9 categories, each with a verified free tier or generous plan.
- **Search & Filter** — Search by name or description, filter by category.
- **Open-Source Data** — The tool database lives in a [separate data repo](https://github.com/jordan-thirkle/freetier-data). Anyone can submit additions via PR or issue.
- **No Account Required** — Completely free, no signup, no tracking.

## Tech Stack

- [Astro](https://astro.build) ^6.1.9
- [Tailwind CSS](https://tailwindcss.com) ^3.4.19

## Getting Started

```bash
npm install
npm run dev
```

The dev server starts at `http://localhost:4321`.

## Build

```bash
npm run build
npm run preview
```
